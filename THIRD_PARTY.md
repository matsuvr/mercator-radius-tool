# Third-party data

## Natural Earth derived data

The bundled basemap file served by the Next.js app is `public/data/world-countries.geojson`.
The legacy PHP source copy remains in `data/world-countries.geojson`.
Natural Earth states that its map dataset is public domain and free for use in any type of project.

Official site:
- https://www.naturalearthdata.com/
- https://www.naturalearthdata.com/about/terms-of-use/
